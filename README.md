# 🌍 A Comprehensive Measure of Well-Being (Human Development Index Prediction)

## 📌 Introduction

The **A Comprehensive Measure of Well-Being** project is a Machine Learning-based web application that predicts the **Human Development Index (HDI)** using key development indicators. The application provides users with an easy-to-use interface to enter input values and obtain the predicted HDI score along with suitable recommendations.

The project is developed using **Python**, **Flask**, **Scikit-learn**, **Pandas**, and **Pickle**, and is deployed on **Render** for online accessibility.

---

## 🎯 Project Objectives

- Predict the Human Development Index (HDI) using Machine Learning.
- Analyze important development indicators affecting HDI.
- Provide users with an interactive web interface.
- Display prediction results instantly.
- Generate recommendations based on the predicted HDI.

---

## 🚀 Features

- User-friendly web interface
- HDI prediction using a trained Machine Learning model
- Four input parameters for prediction
- Personalized recommendations
- Fast prediction using Pickle model
- Responsive Flask application
- Deployed on Render

## ⚙ Installation

### Clone the Repository

```bash
git clone https://github.com/HANSHIKA8899/HUMAN_DEVELOPMENT_INDEX
```

### Move to Project Folder

```bash
cd HUMAN_DEVELOPMENT_INDEX
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
